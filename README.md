# 🐍 Snake Game

A fun and interactive Snake game built with HTML, CSS, and JavaScript!

## Features

✅ Classic snake gameplay
✅ Score tracking
✅ High score saved
✅ Difficulty increases as you play
✅ Smooth controls with arrow keys
✅ Start, Pause, and Reset buttons
✅ Beautiful gradient UI
✅ Mobile responsive

## How to Play

1. **Click "Start Game"** to begin
2. **Use Arrow Keys** (↑ ↓ ← →) to control the snake
3. **Eat the red apples** to grow and earn points
4. **Each apple = 10 points**
5. **Avoid hitting** walls and yourself
6. **Click Pause** to stop the game
7. **Click Reset** to start over

## Game Rules

- 🐍 Snake grows when it eats food
- 🍎 Food appears at random locations
- 📈 Difficulty increases as score increases
- 💥 Game ends if snake hits walls or itself
- 💾 High score is saved in your browser
